Weir provides built-in functions for console I/O, file I/O, and terminal control.

:::note
This page is under construction. Full content coming soon.
:::

## Quick Reference

### Console

| Function | Description |
|---|---|
| `println` | Print value + newline |
| `print` | Print value (no newline) |

### File I/O

| Function | Description |
|---|---|
| `read-file` | Read file contents as string |
| `write-file` | Write string to file |

### Terminal

| Function | Description |
|---|---|
| `term-init` | Enter raw mode |
| `term-restore` | Restore normal mode |
| `read-key` | Read single keypress |

See [Built-in Functions](/stdlib/builtins/) for full signatures and examples.
