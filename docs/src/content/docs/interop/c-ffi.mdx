Weir can call C functions directly using `extern "C"` declarations, with safety enforced via `unsafe` blocks.

## Quick Reference

```lisp
;; Declare external C functions
(extern "C"
  (defn abs ((n : i32)) : i32)
  (defn SDL_Init ((flags : u32)) : i32))

;; Must use unsafe to call them
(unsafe
  (println (abs -42)))
```
